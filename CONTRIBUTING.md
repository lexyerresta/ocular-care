# Panduan Berkontribusi (Contributing Guide)

Terima kasih atas ketertarikan Anda untuk berkontribusi pada **OcularCare**! Kami sangat menyambut ide, perbaikan *bug*, maupun fitur baru dari komunitas.

## 🛠️ Cara Berkontribusi (Pull Request)

Untuk mulai berkontribusi dan mengirimkan *Pull Request (PR)*, silakan ikuti langkah-langkah standar GitHub berikut:

1. **Fork Repositori Ini**
   Klik tombol `Fork` di pojok kanan atas halaman repositori ini untuk membuat salinan proyek ke akun GitHub pribadi Anda.

2. **Clone Repositori Anda**
   ```bash
   git clone https://github.com/USERNAME_ANDA/ocular-care.git
   cd ocular-care
   ```

3. **Buat Branch Baru**
   Buat *branch* baru dengan nama yang mendeskripsikan perubahan Anda (contoh: `fitur-suara-baru` atau `fix-timer-bug`).
   ```bash
   git checkout -b fitur-baru-anda
   ```

4. **Lakukan Perubahan & Commit**
   Lakukan perubahan pada kode yang diperlukan. Pastikan untuk menulis pesan *commit* yang jelas.
   ```bash
   git commit -m "Menambahkan fitur X untuk meningkatkan Y"
   ```

5. **Push ke Branch Anda**
   ```bash
   git push origin fitur-baru-anda
   ```

6. **Buat Pull Request (PR)**
   Kembali ke halaman utama repositori `ocular-care` milik Lexy Erresta. GitHub akan memunculkan banner hijau bertuliskan **"Compare & pull request"**. Klik tombol tersebut, beri deskripsi lengkap tentang apa yang Anda ubah, dan *submit* PR Anda!

## 🐛 Melaporkan Bug & Masalah (Issues)
Jika Anda belum bisa menyumbang kode namun menemukan kendala, *bug*, atau punya ide fitur baru, silakan buka **Issue** di tab **Issues** repositori ini.
